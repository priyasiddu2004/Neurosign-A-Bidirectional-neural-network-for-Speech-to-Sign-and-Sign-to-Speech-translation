# ✋ Sign Language to Text Conversion

## 📌 Project Overview

This project focuses on converting sign language gestures into readable text using computer vision and machine learning techniques. It helps bridge communication gaps between hearing-impaired individuals and others.

---

## 🎯 Objectives

* Detect hand gestures using a camera
* Recognize sign language symbols
* Convert gestures into text output
* Improve accessibility using AI

---

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy
* Machine Learning / Deep Learning 
* TensorFlow 

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/priyasiddu2004/Neurosign-A-Bidirectional-neural-network-for-Speech-to-Sign-and-Sign-to-Speech-translation.git
cd signtotext
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the project:

```bash
python main.py
```
